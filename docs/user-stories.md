# User Stories

## Student

US-01 Submit Academic Issue
As a student,
I want to submit an academic issue online,
so that I can report problems such as missing marks without visiting the registrar's office.

US-02 Track Issue Status
As a student,
I want to track the status of my issue,
so that I know whether the issue is resolved.

US-03 Receive Notifications
As a student,
I want to receive notifications when the status of my issue changes,
so that I stay informed about progress and resolution.

## Registra

US-04 View Submitted Issues
As an academic registrar,
I want to view all submitted academic issues,
so that I can review them and determine the appropriate action.

US-05 Assign Issues to Lecturers
As an academic registrar,
I want to assign reported issues to the appropriate lecturer or department,
so that the responsible staff member can resolve them.

US-06 Monitor Issue Progress
As an academic registrar,
I want to monitor the progress of all reported issues,
so that I can ensure that they are resolved within a reasonable time.

## Lecturer

US-07 View Assigned Issues
As a lecturer,
I want to view academic issues assigned to me,
so that I can review the complaint and investigate the problem.

US-08 Update Issue Status
As a lecturer,
I want to update the status of an issue after investigating it,
so that the registrar and student know the progress.

US-09 Provide Resolution Details
As a lecturer,
I want to provide comments or explanations when resolving an issue,
so that the student understands how the issue was handled.
